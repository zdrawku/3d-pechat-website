# Bulgarian Coiner Cards Images

This folder contains images for the Bulgarian coin holder cards (монетни карти).

## How to Add Images

1. Place product images in this folder (`src/assets/real-images/bg-cards/`)
2. Supported formats: `.jpg`, `.jpeg`, `.png`
3. Required naming convention (front and back for each variant):
   - `variant-1-front.jpg` & `variant-1-back.jpg` - Rectangular card with horizontal flag
   - `variant-2-front.jpg` & `variant-2-back.jpg` - Rectangular card with angled flag (45°)
   - `variant-3-front.jpg` & `variant-3-back.jpg` - Bulgaria map shape with horizontal lines
   - `variant-4-front.jpg` & `variant-4-back.jpg` - Bulgaria map shape with angled lines
   - `variant-5-front.jpg` & `variant-5-back.jpg` - Custom design examples

## Flippable Cards Feature

Each product card on the products page has a carousel-like interface with:
- **Front side image** - Shows the front of the coin card
- **Back side image** - Shows the back of the coin card
- **Navigation buttons** - Left/right chevron buttons to flip between front and back
- **Indicator dots** - Shows which side is currently visible

## Current Status

Placeholder images are currently displayed with green gradients. To show actual product images:

1. Add images following the naming convention above
2. Images will automatically display in the flippable cards
3. Users can click the chevron buttons or use the dots to flip between front and back views
